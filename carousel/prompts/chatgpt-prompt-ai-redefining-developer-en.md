## How to use

1. Go to chatgpt.com and start a new conversation
2. Make sure you are using **GPT-4o** (it has image generation
   via DALL-E built in and can render HTML visually)
3. Paste the prompt below in one message
4. ChatGPT will generate all 8 slides as images
5. Download each image and you are done

---

## Prompt

Create 8 square images (1080x1080px) for an Instagram and
LinkedIn carousel. Each image is one slide. The brand is
t0k1dev — a technical content brand for developers.

**Visual style for all slides:**
- Background: near-black (#0D0D0D)
- Accent color: violet (#7C3AED)
- Main text: off-white (#E5E5E5)
- Secondary text: gray (#737373)
- Font style: modern geometric sans-serif, clean and minimal
- Bold for headlines and key statements
- Regular weight for body text
- No illustrations, no icons, no stock photos
- Typography only — the text is the design
- 80px padding on all sides
- Every slide has "@t0k1dev" small and gray in the
  bottom-left corner
- Every slide has its slide number small and gray in the
  top-right corner
