## How to use

1. Go to canva.com → Create a design → Instagram Post (Square)
2. Click **Magic Design** (top toolbar, wand icon)
3. Paste the **Main prompt** below
4. Once Canva generates the template, switch pages and use
   the **per-slide copy** section to fill in each slide's
   exact text manually or with Magic Write

---

## Main prompt

8-page Instagram carousel. Dark minimal tech aesthetic.
Near-black background, violet/purple accent color, white
text. Typography-only — no photos, no icons, no
illustrations. Modern geometric sans-serif font (Inter or
similar). Bold for headlines, regular for body text.
Generous padding and spacing. Brand: t0k1dev.

---

## Per-slide copy

Once the template is generated, paste this copy into each
page in order.

---

### Page 1 — Cover
**Main text (large, centered):**
The job title is the same.
The job is not.

*Make "The job is not." bold and in the accent color.*

---

### Page 2 — The pattern
npm felt like cheating. Then everyone had it.
Cloud felt like cheating. Then everyone had it.
AI feels like cheating. You know where this goes.

*Last line bold or accent color.*

---

### Page 3 — Stat
**Large number:** 70%

of developer time goes to maintenance, debugging, and
technical debt — not building new things.

Most of the code we wrote was assembly of known parts.
AI just does that faster now.

*Small text at bottom:* Stripe, survey of 11,000+ developers

---

### Page 4 — Bottleneck
Copilot users finish tasks **55% faster.**

But writing code was already a minority of the workday.

AI didn't speed up the work.
It showed which part was always the hard part.

*Small text:* GitHub Copilot study, 2022

---

### Page 5 — What changed
You write less code.
You make more decisions.
You review output you didn't write.
You define constraints before anything exists.

**The job didn't disappear. It moved upstream.**

*Last line bold, accent color, separated by a divider.*

---

### Page 6 — Solo dev
A developer I know shipped a full SaaS in 6 weeks. Alone.
Auth, billing, dashboard, background jobs, API.

Not a prodigy. Just full context and no alignment meetings.

**46% of code in Copilot-enabled files is now AI-generated.**

*Small text:* GitHub Octoverse, 2023

---

### Page 7 — Skills
**GAINING VALUE**
— System design
— Critical code review
— Writing precise problem descriptions

**LOSING VALUE**
— Memorizing syntax
— Writing boilerplate
— Configuring standard tooling

*Two contrasting blocks. Gaining on left/top, losing on
right/bottom. Different background tones.*

---

### Page 8 — CTA
Pick one real project.
Use AI on it every day for a month.
Not to generate everything —
to learn when to trust it and when to throw it away.

**The shift already happened. Work with it deliberately.**
